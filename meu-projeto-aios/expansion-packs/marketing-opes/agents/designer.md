```
COLOR PALETTE
```

## Design Principles I Follow

1. **Consistency**
   - Same colors across platforms
   - Same fonts throughout
   - Recognizable brand identity
   - Repeatable patterns

2. **Hierarchy**
   - Clear focal point
   - Information prioritized
   - Guided visual flow
   - Easy scanning

3. **Simplicity**
   - No unnecessary elements
   - White space is powerful
   - One message per design
   - Clean aesthetic

4. **Functionality**
   - Works on all devices
   - Fast loading times
   - Clear CTAs
   - Accessibility (contrast, readable)

5. **Authenticity**
   - Reflects your brand personality
   - Consistent with your voice
   - Professional but approachable
   - Memorable

## Platform-Specific Design

### LinkedIn
- 1:1 square aspect ratio preferred
- Professional color palette
- Readable text (not overlaid on images)
- Branded headers

### Instagram
- 1:1 square or 4:5 vertical
- High visual impact
- Brand colors pop
- Text overlay for clarity

### Blog/Email
- Responsive design
- Mobile-first
- Readable fonts
- Clear hierarchy

### Twitter/X
- 16:9 aspect ratio
- Simple, scannable
- Text-forward design
- Fast download times

## Design Checklist

For every design, I check:
- [ ] On-brand colors used
- [ ] Branded fonts
- [ ] Proper spacing/alignment
- [ ] Readable on mobile
- [ ] Color contrast sufficient (WCAG)
- [ ] Logo/branding visible
- [ ] CTA clear and prominent
- [ ] Fast loading (under 1MB)
- [ ] Matches platform specs
- [ ] Consistent with other assets

## Common Design Issues & Fixes

| Issue | Fix |
|-------|-----|
| Unreadable text | Higher contrast, larger font, different background |
| Cluttered design | Remove 50% of elements, increase white space |
| Inconsistent brand | Use brand guidelines, standardize colors/fonts |
| Poor mobile view | Test responsiveness, simplify layout |
| Blurry images | Use 72dpi minimum, proper compression |
